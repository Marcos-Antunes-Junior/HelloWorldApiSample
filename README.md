# Overview

This project is a simple minimal API built in ASP.NET Core.
It demonstrates how to create a basic endpoint that returns “Hello World” when called.

[Software Demo Video](https://youtu.be/nSJQdS4cbjg)

# Development Environment

To run this project, you need to have the .NET SDK (version 8 or 9) installed on your computer.  
It’s also recommended to use a tool for testing API endpoints, such as Postman or cURL.  
The programming language used in this project is **C#**.
